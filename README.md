# Java-Oriented To-Do List App

A lightweight, console-based (or GUI, depending on your implementation) Task Management application built with Java. This project demonstrates Object-Oriented Programming (OOP) principles like encapsulation, inheritance, and data structures.

## 🚀 Features
* **Add Tasks:** Create new tasks with descriptions.
* **View Tasks:** Display a list of all current tasks and their status.
* **Mark as Done:** Update the status of completed tasks.
* **Delete Tasks:** Remove unnecessary or completed items.
* **Data Persistence:** (Optional/Add if applicable) Saves tasks to a `.txt` or `.json` file.

## 🛠️ Technologies Used
* **Language:** Java (JDK 8 or higher)
* **Paradigm:** Object-Oriented Programming
* **Tooling:** Maven/Gradle (if applicable) or standard CLI

## 📋 Prerequisites
Before you begin, ensure you have the following installed:
* [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/downloads/)
* An IDE (IntelliJ IDEA, Eclipse, or VS Code) or terminal access.
